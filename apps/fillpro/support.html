Local data note

Local data is self-managed.

FillPro stores your profiles on your device. I cannot remotely inspect them, export them, or delete them for you.

  • Delete profiles directly in FillPro.
  • Uninstall the extension to remove extension-managed storage.
  • Delete exported files separately if you saved them outside the extension.

Fast start

Get to the first real fill quickly.

  1. Step 1: Create a profile in the FillPro popup.
  2. Step 2: Add custom fields, smart rules, or local uploads only if you need them.
  3. Step 3: Open a normal web form and use Fill Page, the on-page picker, or Ctrl+Shift+F.

Smart rules

Use them for awkward labels.

  • Plain text does a contains match, for example work email.
  • Regex handles harder cases, for example /best.*contact/i.
  • @field references reuse saved values such as @email or @company_size.

Local uploads

No fixed FillPro file-size cap.

  • Supported formats: PDF, DOC, DOCX, TXT, RTF, ODT, and Pages.
  • FillPro does not set a fixed per-file upload limit.
  • Practical browser, device, and destination-site limits still apply.
  • Use clear target patterns like resume, cv, candidate resume, or /cover.?letter/i.

Known limits

Known browser and site limits.

  • Sites behind CAPTCHA, bot checks, or human verification must be cleared manually first.
  • Cross-origin embedded forms can stay unreachable under Chrome's privacy-first activeTab model.
  • Browser-native PDF tabs are not fillable like normal HTML forms.
  • Payment cards, CVC, passwords, SSNs, tax IDs, and one-time codes are skipped by design.

Expectation check

What FillPro is not trying to be.

  • Not a password manager, payment autofill tool, or sensitive-data vault.
  • Not random fake-data generation. QA testers can create dedicated test profiles instead.
  • Not automatic page-load filling. You stay in control with Fill Page, the picker, or Ctrl+Shift+F.
  • Not cloud sync. Profiles stay local; Pro import/export can move profiles manually.
  • Not guaranteed to control custom dropdowns or date pickers when a site hides the real field.

Troubleshooting

When a form is close but not quite there.

  • If nothing fills, wait until the real form is visible, then try again.
  • If the wrong field was matched, add a smart rule for that site-specific label.
  • If a multi-step form changes after the first step, fill again after the next step appears.
  • If a custom dropdown or date picker does not fill, select it manually and report the site so it can be reviewed.
  • If an upload field is missed, make the file pattern more explicit.
  • If the page is dynamic, try the on-page picker after the form fully appears.

Use cases

Where FillPro fits best.

  • Job applications and candidate forms
  • Account creation and onboarding flows
  • Lead capture, demo request, and intake forms
  • Checkout contact and shipping sections

Billing / Pro

What changes when you upgrade.

  • Free mode includes 3 saved profiles.
  • Pro adds more saved profiles, import/export, and duplication.
  • Payments are handled by ExtensionPay and Stripe.
  • Free-use profile data still stays local on your device.
