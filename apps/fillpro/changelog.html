Version 1.0.0

What shipped in the first public release

Core flow

  • One-click Fill Page from the popup for normal web forms.
  • On-page profile picker so you can fill without breaking focus.
  • Custom fields and smart rules for awkward labels and site-specific matching.
  • Local file upload matching for resume, CV, and cover-letter style inputs.

Privacy from day one

  • Profile data stays local on your device.
  • No broad production host permissions.
  • Sensitive fields are intentionally skipped.
  • Optional feedback diagnostics are only prepared when you ask for them.

Small but important details

  • A lighter first-run flow that gets you to a real fill faster.
  • Support and privacy pages written for actual use, not filler.
  • Free mode works without an account.

Still not supported

What FillPro will not try to fake.

  • CAPTCHA, anti-bot checks, and verification walls
  • Cross-origin payment iframes
  • Browser-native PDF viewers
  • Pages that block access before a normal form is even reachable

If a site blocks one of those paths, FillPro says so plainly instead of pretending it worked.
